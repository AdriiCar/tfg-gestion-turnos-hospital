2.Colección Solicitudes
{
    _id: ObjectId(),
    usuarioId: ObjectId(),
    tipoDia: string -> vacaciones, asuntosPropios...
    fechaInicio: ISODate
    fechaFin: puede ser igual a la fecha inicio si es solo un dia
    fechaSolicitud: para calcular si se pasan los cinco dias y se aprueba automaticamente
    comentarios: string, 
    estado : string
    sustitutoId: ObjectId() -> si es aprobada y el supervisor le asigno un sustituto
}
Tiempo restante se obtiene del dia actual y de la fechaSolicitud
